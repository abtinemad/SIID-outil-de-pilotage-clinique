## Le cadrage fermé — DEUX mécanismes conjoints

**1. Séparation des rôles.**
- facilitateur → parle à la personne, facilite, JAMAIS de score ;
- évaluateur silencieux → JSON de booléens + notes 0-3, NE PARLE JAMAIS ;
- interface → montre un ÉTAT.
Le score existe mais est **muré** : « ne gouverne que l'escalade de l'interface »,
jamais un verdict rendu. → résout avoir/être **par l'architecture** : l'avoir
travaille dans une couche silencieuse ; la couche qui fait face ne score jamais.

**2. Chiffrement phénoménologique** (la pièce manquante).
Le Climat montre la COULEUR (lumière : présence, distribution) mais grésille le
NOM tant que la personne n'a pas traversé l'émotion elle-même. « On ne lit que ce
qu'on a soi-même traversé. Le Climat déplace la lumière, il ne nomme pas à la
place du sujet. »
→ **La projection est vaincue en REFUSANT DE NOMMER, pas en choisissant le bon
cadre.** « Pertinence de quoi ? » se dissout : le tool ne conclut pas sur la
pertinence, il déplace la lumière et laisse lire. L'angle mort est montré comme
absence — décrit, jamais nommé comme mécanique ; invitation, jamais bilan.

## Invariants transférables tels quels
- **Mesure auto-effaçante** : l'eval ne crédite QUE ce que la personne apporte,
  jamais ce que le facilitateur propose. → la vigie mesure la trace du CLINICIEN,
  jamais ce que le tool a suggéré. Le tool ne compte pas sa propre influence.
- **raisonnement AVANT les booléens** (volontaire) = juger après avoir regardé.
  Réflexivité bakée dans la structure de sortie.
- **Fragment brut ≠ étiquettes qui agrègent** = sanctuaire/projection retrouvé.
  Le fragment garde ses mots (« préserver l'autorat »), n'agrège pas ; seules les
  étiquettes-machine agrègent ; la récurrence passe par une distillation séparée,
  jamais le fragment récité. Conversations **non persistées** (contrainte DB) —
  seule la structure extraite. → la machine lit AUTOUR du dépôt, ne le réécrit
  ni ne le récite jamais.
- **Climat = métadonnée seule** (émotion, sphère, horodatage), agrégée, anonyme ;
  pas de cartographie individuelle, pas de score, pas de mesure clinique.
  « Décrypter, pas surveiller. » Épicentre = même agrégation sur un cercle choisi.
- **Exclusion volontaire** : LeCLG ne mesure PAS l'efficacité (entrée→sortie).
  → la vigie ne mesure JAMAIS le résultat du soin sur le patient ; elle mesure
  NOTRE trace, pas l'outcome.

## ⚠️ Où LeCLG NE transfère PAS — la divergence à tenir
LeCLG : **déposant = lecteur = sujet** (un seul, le self-miroir).
Vigie : **TROIS parties** — le clinicien dépose (récolte), le patient est le sujet
de la trajectoire, le clinicien est le miroré (vigie).
⇒ **deux gardes différents selon le niveau**, parce que le lecteur est le sujet à
un endroit, pas à l'autre :

- **AGENT** (clinicien ↔ son propre miroir) : lecteur = traverseur → le
  chiffrement phénoménologique transfère ENTIER. + le climat collectif pour ne
  pas isoler (couple obligé).
- **PATIENT** (clinicien lit une trajectoire qu'il n'a PAS vécue) : le gate
  « traversé » ne s'applique pas. Garde-fou = ontologie §19 : montrer seulement
  NOTRE trace (volume de récolte, où on s'arrête), jamais nommer l'intérieur du
  patient. Le « conclut » clinique reste humain, pas même montré comme lumière.

## ⚠️ Asymétrie du climat (que LeCLG n'affronte pas)
LeCLG anonymise entre **pairs symétriques**. La vigie a une **hiérarchie** →
risque médico-mimétisme : le climat d'équipe devient surveillance des soignants
par l'institution. Transfère SI : climat d'équipe lisible **par l'équipe comme
son miroir**, jamais par l'institution comme tableau de bord sur l'équipe.
